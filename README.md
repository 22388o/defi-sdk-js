# defi-sdk

A JS client for interacting with [Zerion API](https://github.com/zeriontech/zerion-api)

## Install

```sh
npm install defi-sdk
```

## Getting Started

```js
import { client } from "defi-sdk";

client.configure({ url: endpoint, apiToken: API_TOKEN });
```
